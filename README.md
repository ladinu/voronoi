voronoi
=======

Simple [voronoi](http://en.wikipedia.org/wiki/Voronoi_diagram) diagram generator written in go in an afternoon.

## Generate Diagram
JPEG will be written to `out.jpg`
```sh 
$ go run voronoi.go
```
